# New User Onboarding

## Goal

A new player should be able to create or join a room without an account.

## Entry Points

- Create a room from the home/setup flow.
- Join a room from an invite link.
- Join a room by entering a room code.

## Identity

Players use guest identity for the MVP.

- Display name is required.
- Player color uses the fixed palette or supported custom color input.
- Active state must not rely on color only.

## Success State

After room creation or join, the player reaches lobby/setup or active match state without manual protocol navigation.
