# Lobby

## Purpose

Lobby lets players gather, choose supported match options, and start play.

## User Behavior

- Host can create a room.
- Players can join by invite link or room code.
- Players can choose name and color.
- Lobby can select mode: Classical or Party.
- Setup/lobby board preview is display-only and must not create game state.

## Classical Defaults

- 2 players.
- 15 x 15 board.
- 8 rack slots.
- Standard tile bag and rules.

## Party Options

Party mode can support more players, larger odd board sizes, tile bag scaling, custom time controls, and optional skill nodes/actions.
