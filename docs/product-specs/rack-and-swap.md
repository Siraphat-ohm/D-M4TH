# Rack And Swap

## Rack

Rack always has 8 slots. Empty placeholders remain visible.

Rack tiles must stay readable without overpowering the board.

## Placement

Desktop supports drag/drop and click placement. Mobile and tablet primarily support selecting a rack tile and tapping a board cell.

Draft board tiles return to rack on double click or double tap.

## Actions

The action bar includes:

- Play
- Swap
- Pass
- Recall

Play uses active player color only when the current player can act. Swap, Pass, Recall, and disabled actions stay muted and neutral.

## Swap Rule

Swap exchanges 1 to 8 selected rack tiles with the tile bag and skips the turn. Swap is unavailable when the bag has 5 or fewer tiles.
